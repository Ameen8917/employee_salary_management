# Salary Management System

A web-based Salary Management System built as part of the technical assessment.

## Objective

Provide HR managers with a centralized application to manage salary information for approximately 10,000 employees and generate salary insights.

## Planned Tech Stack

### Backend
- Node.js
- Express
- TypeScript
- Sequelize
- SQLite

### Frontend
- React.js
- TypeScript
- Tailwind CSS
- shadcn/ui
